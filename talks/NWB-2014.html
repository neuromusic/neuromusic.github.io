Experimental Events

The Experimental Events data type consists of times of events, along with values that correspond to the times. This data type can be used to describe: stimuli, trials or sweeps, time intervals, behaviors, and other events or conditions that occur during the experiment.

INCF requirements for storing electrophysiology data (PDF, 2014-11-03)

Event Requirements

  • Starting time of event monitoring (earliest time an event could be detected)
  • Ending time of event monitoring (latest time that an event could be detected)
  • Description of the events
  • Sequence of event times
  • Property values needed to describe each event.
